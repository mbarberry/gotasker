# GoTasker

## Overview

Built with a Go AWS Lambda function backend, this application allows users to maintain an online todo list.

## Features

- **Go Lambda Backend** – Serverless Go function handles API operations.
- **Online Todo List** – Add, update, and delete tasks from anywhere.
- **Persistent Storage** – Todos are saved and managed in the MongoDB database.
- **User-Friendly Interface** – Simple and intuitive web interface for managing tasks.
